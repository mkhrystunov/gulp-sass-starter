#### Installation
`npm install`

#### Starting developing
`npm run watch`

#### Usage
Add your scss files to `./src/sass` directory.
Compiled css file will appear in `./dist` folder.


#### SASS or SCSS
If you want to use `*.sass` format instead of `*.scss` change
lines 8 and 15 of `gulpfile.js` from `'./src/sass/**/*.scss'`
to `'./src/sass/**/*.sass'`
